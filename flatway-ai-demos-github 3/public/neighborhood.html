Comprendre un quartier avant de visiter

Un acheteur qui ne connaît pas un arrondissement doit aujourd'hui chercher lui-même sur plusieurs sites. Ici : un résumé en langage clair du quartier, et un calcul séparé du temps de trajet vers un point de référence — deux choses différentes, l'une écrite par l'IA, l'autre un simple calcul.

Données illustratives, deux mécaniques différentes. Les données de quartier ci-dessous (transport, écoles, commerces) sont des exemples — pas une vraie base de données de quartier. Ce qui est réel : le résumé est écrit par l'IA à partir uniquement de ces données fournies (jamais d'invention). Le calculateur de trajet plus bas n'est pas de l'IA — c'est une simple table de temps approximatifs, comme un tableau Excel.
